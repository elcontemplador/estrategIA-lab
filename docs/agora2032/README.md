# Portal ciudadano Ágora 2032

Demo narrativa estática vinculada a las historias veraniegas de `estrategIA`.

## Experiencias

- revisión del expediente ficticio de Samira Haddad;
- reparto de cómputo durante una emergencia;
- archivo temático de los nueve relatos;
- perfil final construido a partir de las decisiones del visitante.

## Principios

- no utiliza API ni modelos de IA;
- no recopila datos personales;
- no envía información a ningún servidor;
- funciona como sitio estático en GitHub Pages;
- distingue expresamente la simulación narrativa de un servicio público real.

## Ejecución local

Puede abrirse directamente `index.html`, aunque para las pruebas automatizadas es preferible servir la carpeta mediante un servidor HTTP local.
